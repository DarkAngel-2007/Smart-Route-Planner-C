# 🚀 Smart Route Planner (C)

## 📌 Overview
The Smart Route Planner is a graph-based project implemented in C that helps in finding:
- The shortest path between cities
- The optimal route to visit all cities with minimum cost

This project demonstrates the use of important algorithmic techniques like **Greedy Method** and **Dynamic Programming** in real-world scenarios such as navigation and route optimization.

---

## 🧠 Algorithms Used

### 🔹 Greedy Algorithm – Dijkstra
- Used to find the shortest path from a source city to all other cities
- Selects the nearest unvisited city at each step
- Efficient for pathfinding in graphs

### 🔹 Dynamic Programming – Traveling Salesman Problem (TSP)
- Finds the minimum cost to visit all cities exactly once and return to the starting point
- Uses **bitmasking and memoization** to optimize calculations
- Avoids recomputation of subproblems
